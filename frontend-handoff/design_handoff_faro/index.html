Faro · Handoff de diseño

Kit de pantallas HTML estáticas para que Claude Design produzca el styles.css definitivo. Lee el README.md antes de diseñar.

Pantallas

  • 01 — Home (landing) — Página de bienvenida con enlace a la cascada de Aula.
  • 02 — Cascada de Aula — Demo: selector de materia/OA + resultados completos (validación, unidad, clases, prueba, deck).
  • 03 — Planificación · Formulario — Paso 1: elección de formato, asignatura, nivel, OA, unidad y docente.
  • 04 — Planificación · Generando — Estado de espera mientras el worker genera la planificación.
  • 05 — Planificación · Revisión (HIL) — Pantalla de revisión: campos editables + 3 paneles de generación (prueba lista, PPT idle, guía idle).
  • 06 — Revisión HIL (global) — Superficie de revisión docente: lista de pendientes + detalle abierto con gates + acciones.
  • 07 — Producción asíncrona — Flujo de producción: selección de planificación/unidad + job en curso + borradores generados.

Instrucciones rápidas

  1. Abre README.md (el prompt para Claude Design) y léelo completo.
  2. Diseña styles.css usando las clases del índice en ese archivo.
  3. Verifica tu diseño abriendo cada pantalla directamente en el navegador (sin build).
  4. No modifiques el HTML — solo styles.css.
